A powerful, real-time wordlist generator with a stunning UI and streaming capabilities. Generate custom password lists, brute-force dictionaries, or word combinations on the fly with zero memory overhead.

![Wordlist Pro Demo](https://via.placeholder.com/800x400?text=Wordlist+Pro+Screenshot)

## ✨ Features

- **Real-time Streaming** - Generate and download files instantly without memory limitations
- **Customizable Combinations** - Mix words, numbers (including ranges), and special characters
- **Smart Word Processing** - Automatically generates lowercase, uppercase, and capitalized variants
- **Number Range Support** - Use ranges like `1-100` or specific numbers like `123,456,789`
- **Interactive UI** - Beautiful dark theme with animated shooting stars effect
- **No Memory Limits** - Streams directly to browser, handles billions of combinations
- **Fast Performance** - Built with Spring Boot's StreamingResponseBody
- **Responsive Design** - Works on desktop and mobile devices
